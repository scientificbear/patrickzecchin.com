Functional data clustering – an application to mobile network data

Portfolio

Summary

The purpose of this thesis is to illustrate a set of techniques able to describe and to group a given set of data and to show a real-life implementation of these methods. Since the dataset consists of curves, some precautions and expedients need to be considered in order to deal with elements that lie in an infinite-dimensional space.

The topic of this paper arose during an internship programme in TIM, probably the most important Italian telecommunications company. The content of this thesis is strictly confidential, since the following topics are protected by law as secret.
